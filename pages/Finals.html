    Outputing Data

      There are two ways to output the data to the screen:

    • echo : echo has no return value and can take many or multiple parameters.
    • print : print has a return value of 1 and can take only one parameter.

        • This is how you output a String using echo and print:

      • $echo "Hello World";
      • print "World, Hello";

        This is how you output a variable using echo and print:

      • echo $variable;
      • print $number1 + $number2;


    Variable Scopes

    • local scope are variable declared inside a function and can be accessed only within the function,
      example: $x = 10;
    • global scope are variable declared outside a function and can be accessed only outside the function,
      example: global $x = 5;
    • static scope are variables that retains the value even if the page is reloaded or repeated,
      example: static $x = 5;

    Servlet Technologies is used to create web applications.

    Servlet is a java proggraming language that is easy to use and it is scalable, robust and object-oriented.

    Before servlets are introduced, CGI (Common Gateway Interface) is used.
    It is also a proggraming language that is used to create a web applications.

    • Java Servlets

      • Java objects that are intended to play the role of server components in Client-Server communications.
      Java program that give capabilities or functionalities to run a server.
      Handles clients request by generating responses to such request.

    • Servlet LifeCycle

      1. Instantiation => constructor call..
        • creates an instance of the servlet to service client requests.
      2. Initialization => init() method..
        • invoked only once and is intended for any startup initialization code for the servlet.
      3. Request Handling => service()method..
        • the term service method is used for any method in a servlet class that provides a service to a client.
          • Methods:
          • doGet()
          • doDelete()
          • doOptions()
          • doPost()
          • doPut()
          • doTrace()
      4. Destruction => destroy() method..
        • invoked before the servlet instance is 'unloaded' and is intended for any 'housekeeping'.


      HTTP Servlet

    • Java servlet used to handle HTTP request and HTTP response.
    • Java servlet used to handle HTTP requests and generate HTTP responses.
    • Hosted in a 'web container', which is a component in a Java 'application server' (eg. Apache Tomcat, Oracle Glassfish, Red Hat JBoss/Wildfly etc.).

      • Web Container

    • a part of web server that manages the components like servlets.

      • HTTP Servlet Request

    • object representation of HTTP request.
    • A request contains data passed between a client and the servlet.
    • HTTP servlets are passed an HTTP request object, HttpServletRequest, which contains the request URL, HTTP headers, query string, and so on.

      • HTTP Servlet Response

    • object representation of HTTP response.
    • HTTP response objects, HttpServletResponse, have fields representing HTTP headers.
